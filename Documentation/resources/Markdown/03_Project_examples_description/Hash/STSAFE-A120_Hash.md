# STSAFE-A120 Hash {#STSAFE-A120_Hash}

This example demonstrates hash computation using the STSAFE-A120 secure element.  

## Example Overview

The Hash example application illustrates how to compute a SHA-256 hash message using platform crypto library and using STSAFE-A120 device.

## Example Flowchart

The following flowchart outlines the main steps performed by the hash computation example application:

@startuml "ECDH Example flowchart" width=5cm
:MAIN;
:Initialize application terminal (baudrate = 115200);
:Display example title and user instructions;
:ret = <b>stse_init</b>;
if (ret != STSE_OK) then (Yes)
	:Display error message;
	note right: Infinite loop
	-[hidden]->
	detach
else (No)
	:ret = <b>stse_platform_hash_compute</b>;
	if (ret != STSE_OK) then (Yes)
		:Display error message;
		note right: Infinite loop
		-[hidden]->
		detach
	else (No)
		:Display the computed hash message using platform crypto library;
		:ret = <b>stse_compute_hash</b>;
		if (ret != STSE_OK) then (Yes)
			:Display error message;
			note right: Infinite loop
			-[hidden]->
			detach
		else (No)
			:Display the computed hash message using STSAFE-A120 device;
			:Compare both messages;
			if (both messages equal) then (Yes)
				:Success!;
				stop
			else (No)
				:Comparison error;
				note right: Infinite loop
				-[hidden]->
				detach
			endif
		endif
	endif
endif
@enduml
